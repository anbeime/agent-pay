## 2. 核心组件说明

### 2.1 Frontend Layer
| 组件 | 技术栈 | 功能 |
|------|--------|------|
| Dashboard Panel | React + Chart.js | 实时收益展示、资产配置可视化 |
| Deposit/Withdraw | wagmi + viem | 用户资金存取交互 |
| Strategy Selector | React Components | 策略选择、风险偏好设置 |
| Analytics Panel | D3.js | 历史收益分析、风险评估 |

### 2.2 AI Agent Layer
| Agent | 职责 | 输入 | 输出 |
|-------|------|------|------|
| Data Collector | 收集市链上数据 | Protocol APIs, Price Feeds | 结构化市场数据 |
| Analysis Agent | 分析收益与风险 | 市场数据 + 用户偏好 | 风险评估报告 |
| Decision Agent | 制定投资决策 | 分析报告 | 最优配置方案 |
| Execution Agent | 执行链上操作 | 配置方案 | 交易签名 |

### 2.3 Smart Contract Layer
| 合约 | 功能 | 关键方法 |
|------|------|----------|
| YieldVault | 资金托管 | deposit(), withdraw(), harvest() |
| StrategyManager | 策略管理 | addStrategy(), rebalance() |
| AgentController | Agent权限控制 | authorize(), execute() |
| RiskController | 风险管理 | assessRisk(), emergencyExit() |

## 3. 数据流设计

```
用户操作                    Agent处理                    链上执行

[Deposit] ---------> [验证金额] ---------> [YieldVault.deposit()]
                          |                          |
                          v                          v
                    [风险评估]              [铸造Share Token]
                          |
                          v
                    [选择策略] ---------> [StrategyManager.allocate()]
                          |
                          v
                    [执行配置] ---------> [Protocol.deposit()]

[监控循环] <----------------------------------- [链上事件]
     |
     +---> [收益监控] --> [分析比较] --> [是否再平衡?]
                                            |
                                     Yes    |    No
                                     +------+------+
                                     v             v
                              [生成再平衡]    [继续监控]
                              [交易]
                                     |
                                     v
                              [执行再平衡] --> [StrategyManager.rebalance()]
```
