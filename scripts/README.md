# Rebel AI Hackathon - 脚本目录

本目录包含项目开发所需的自动化脚本。

## 脚本列表

### call_skill_template.py
技能调用模板，用于自动化调用各种技能进行开发。

**使用方法:**

```bash
# 执行所有任务
python call_skill_template.py --all

# 执行指定任务
python call_skill_template.py --task 1
```

**功能:**
- 任务状态管理
- 技能调用封装
- 进度记录
- 文档生成
